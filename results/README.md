# Results / Resultados

This directory contains the logs and artifacts generated during the chaos engineering experiments. / Este diretório contém os logs e artefatos gerados durante os experimentos de engenharia de caos.

## Contents / Conteúdo
- `logs/`: Time-stamped execution logs. / Logs de execução com registro de data/hora.
- `pcap/`: Network traffic captures. / Capturas de tráfego de rede (.pcap).
- `post-mortem/`: Analysis of the system's behavior under stress. / Análise do comportamento do sistema sob estresse.
